### Ombre e Bordo Arrotondato

Creare un index.html con i normali header
Aggiungere un titolo h1 e almeno un immagine dal sito https://free-images.com/ usando <img>
Crea un file CSS esterno styles.css
Collegare il CSS all'html con link
Applica ombre e bordi arrotondati agli elementi principali usando text-shadow per migliorare l'estetica.
